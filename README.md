## Welcome to e-Larc-api GitHub Pages

Larc is a French acronym, which means Software to help in the search and collection of skills. In English, it would be Srash.

Its main purpose is to be able to generate report cards for students in an IB International School (IBO.org).

In short, this multi-tier architecture software is based on a Postgresql database of the same name and on an API provider created using Django and Django Rest Framework.

This repository only contains the database and API generator part.
